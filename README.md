# Dynacademy — AI-Powered Education & Career Guidance Platform

A comprehensive AI-powered platform designed to help South African high school learners transition successfully into universities, TVET colleges, bursary opportunities, and future careers.

## Features
- AI Career Guidance: recommendations based on subjects, marks, interests, and personality
- Live APS Score Calculator with university eligibility tracking
- University & TVET College Matching
- AI Bursary Finder with 5,000+ funding opportunities
- AI Study Assistant: summaries, quizzes, study plans
- Student Wellness Module with mood tracking
- AI CV & Motivation Letter Generator
- Multilingual support for all South African languages
- Responsive design (mobile, tablet, desktop)

## Tech Stack
React | Vite | Tailwind CSS | Framer Motion | Firebase/Supabase | OpenAI API | Chart.js | Recharts
